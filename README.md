SFXScout v0.9.0 — Friends & Family Preview

This repository hosts public release builds of SFXScout for friends & family testing. Source code is not included here.

Install
1. Download the latest .zip from Releases
2. Unzip and drag SFXScout.app to Applications
3. First launch: right-click → Open
4. Add your SFX folder(s) and let indexing complete
